# Музыкальные подложки

Положи сюда royalty-free треки `.mp3` / `.m4a` — они появятся в выпадашке UI
(раздел «2 · Рендер» → выбор музыки) и будут подмешаны ТИХО под озвучку (ducking ~-18dB).

Источники без копирайта: YouTube Audio Library, Pixabay Music, Free Music Archive (CC0/CC-BY).
Не клади треки с защитой авторских прав — это для паблик-постинга.
